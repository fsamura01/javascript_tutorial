‚ùì The Question

How many ways can we pick THREE 2's to form the triplet (2, 2, 2)?

We need to choose 3 positions from 4 available positions where all have the value "2".

üìê The Combination Formula C(n, 3)
C(n, 3) = n! / (3! √ó (n-3)!)
C(n, 3) = n √ó (n-1) √ó (n-2) / (3 √ó 2 √ó 1)
C(n, 3) = n √ó (n-1) √ó (n-2) / 6
Where n = freq[x] (the number of times value x appears)

ü§î Why Divide by 6?

The Logic:

If we were picking with order mattering (permutations), we'd have:

First pick: 4 choices
Second pick: 3 remaining choices
Third pick: 2 remaining choices
Total ordered ways: 4 √ó 3 √ó 2 = 24

BUT! Order doesn't matter for us. The triplet (2‚ÇÅ, 2‚ÇÇ, 2‚ÇÉ) is the same as (2‚ÇÉ, 2‚ÇÅ, 2‚ÇÇ).

How many ways can we arrange 3 items? 3! = 3 √ó 2 √ó 1 = 6 ways

Combinations = Permutations / Arrangements
C(4, 3) = 24 / 6 = 4
